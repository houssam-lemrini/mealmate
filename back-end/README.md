# Back-End

This directory contains the **backend** code for the project, built with **FastAPI**.

### Purpose
All server-side logic, API routes, and database connections will be implemented here.

### Notes
- Framework: [FastAPI](https://fastapi.tiangolo.com/)
- Entry point: `main.py`
- Use virtual environment for dependencies (`venv` or `poetry`)
- Run with:
  ```bash
  uvicorn main:app --reload
### Requirements

Install dependencies using:
```bash
pip install -r requirements.txt
